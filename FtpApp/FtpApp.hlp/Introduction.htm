FtpApp

Introduction

FTP App is an application designed to perform FTP operations to a website that requires encrypted login but unencrypted ftp operations. It uses OpenSSL to perform the encrypted operations.

The application maintains a database for each web site. Each file in the website is represented in the database file. In particular the size and the modification date of the local and remote files is kept. Then when changes are made to the local files, the application compares the sizes and dates to determine if the local file needs to be sent to the database or the remote file needs to be copied back to the local directory. If the file was deleted from the web site since the database was created or since the last update then the application will ask if the file should be deleted from the web site.

All the operations are explained in detail on the Operations Page. here is the list of operations:

  • Create Web Site Database
  • Login to Web Site Database
  • Compare Current Web Site To Database
  • Confirm One or More Updates
  • Save Database
