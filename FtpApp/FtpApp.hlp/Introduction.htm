FtpApp

Introduction

FTP App is an application designed to perform FTP operations to a website that requires encrypted login but unencrypted ftp operations. It uses OpenSSL to perform the encrypted operations.

The application maintains a database for each web site. Each file in the website is represented in the database file at the time of the last update. In particular the size and the modification date of the local file is kept in this database. This is done because the web hosts use different operation systems with different ways to measure file size and they sometimes change the dates of the files.

All the operations are explained in detail on the Operations Page. here is the list of operations:

  • Create Web Site Database
  • Pick a Web Site from the list contain in the application
  • Compare Current Web Site To version at the last update
  • Pick a Web Site to download from the web host
  • Select Files and Confirm the Operation Selected (Put or Get)
